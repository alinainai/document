
## 1、方法和字段的符号表示

<img width="203" alt="image" src="https://user-images.githubusercontent.com/17560388/187613203-66586141-1320-49b2-a5f7-41a71d705b6e.png">

|  符号   | 含义  |
|  :----: | ----  |
| -  | 表示private |
| +  | 表示public   |
| #  | 表示protected  |
| ~  | 表示default,也就是包权限 |
| _  | 加下划线表示static   |
| 斜体  | 表示抽象 |
