3、关闭键盘
```dart
class HideKeybUtils{
  /// 关闭键盘并保留焦点
  static Future<void> hideKeyShowfocus() async{
    await SystemChannels.textInput.invokeMethod('TextInput.hide');
  }
  /// 关闭键盘并失去焦点
  static Future<void> hideKeyShowUnfocus() async{
    FocusManager.instance.primaryFocus.unfocus();
  }
} 
```
